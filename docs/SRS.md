# Software Requirements Specification (SRS)

## Improved Requirements

# The system shall allow users to complete the registration process within 3 minutes without external help.
# Requirement 2:
# The system shall store student records in an encrypted database using role-based access control.
# Requirement 3:
# The system shall load the dashboard page within 2 seconds under normal network conditions.
# Requirement 4:
# The system shall allow users to register using a valid email address and password.
# Requirement 5:
# The system shall allow registered users to log in using correct credentials and manage their profile information.
# Requirement 6:
# The system shall send email notifications to users after successful registration and password reset
